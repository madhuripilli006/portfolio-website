# Full-Stack Personal Portfolio Website

## Description
A responsive personal portfolio website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features
- Responsive UI
- Contact Form
- Backend API
- MongoDB Integration
- Modern Design

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Deployment
Frontend: GitHub Pages / Vercel  
Backend: Render
